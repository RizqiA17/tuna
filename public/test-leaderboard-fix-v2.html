🔍 Root Cause Analysis

🚨 Masalah yang Ditemukan:

Root Cause: Di method showAppropriateContent(), ketika menangani leaderboard-content, kode memanggil this.showLeaderboard() yang kemudian memanggil this.loadCurrentScenario() di dalamnya.

// Masalah di showAppropriateContent(): if (this.currentScreen === 'leaderboard-content') { // ... if (this.teamData) { this.showLeaderboard(); // ❌ Ini memanggil loadCurrentScenario() } } // Di loadCurrentScenario(): this.currentScreen = 'scenario-content'; // ❌ Ini meng-override leaderboard-content!

🔧 Solusi yang Diterapkan:

  1. Buat Method Baru: loadLeaderboardData() yang hanya memuat data tanpa mengubah currentScreen
  2. Ganti Pemanggilan: Ganti this.showLeaderboard() dengan this.loadLeaderboardData()
  3. Preserve Screen State: Pastikan currentScreen tetap 'leaderboard-content'
